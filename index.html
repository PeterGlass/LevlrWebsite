FEATURES

Levlr was designed based on a want for having background audio that does not distract from primary audio sources

Main Volume

Adjust individual source's main volume similar to the native volume mixer

Threshold Cutoff

Set the threshold level that will trigger lower ranked sources to go to a dropped volume state

Drop Volume

Adjust the volume that individual sources will drop to when higher rank sources reach their threshold

Source Selection

Choose which audio sources that you want levlr to monitor and control

Source Layer

Configure the rank of individual sources. These ranks determine how sources affect each other

Drop Delay

Configure how long a individual source will will take to switch from a dropped to non-dropped state
